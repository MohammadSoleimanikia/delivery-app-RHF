# 🚀 React Hook Form Practice Project

A comprehensive practice project demonstrating advanced form handling in React using **React Hook Form (RHF)** with **Zod validation**, **TypeScript**, and **Tailwind CSS**.

## ✨ Features

- ✅ **React Hook Form** - Efficient form state management and validation
- 🔒 **Zod Schema Validation** - Type-safe form validation with custom rules
- 📝 **Nested Form Fields** - Handle complex nested form structures (address fields)
- 🎨 **Reusable Components** - Custom `Input` and `Select` components with error handling
- 🎯 **Multiple Validation Modes** - On-change validation with Zod resolver
- 🧩 **Form Context** - Using `FormProvider` to avoid prop drilling


## 🛠️ Tech Stack

- **React** - UI framework
- **React Hook Form** - Form management
- **Zod** - Schema validation
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Vite** - Build tool (assumed)
